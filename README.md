## Installation

CCUtils is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
# 设置源
source 'https://github.com/buvtopcc/CCSpecs.git'
pod 'CCUtils'
```
